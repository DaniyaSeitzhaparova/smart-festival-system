# Smart Festival Management System

**Project goal:**  
To integrate technology and sustainability for large-scale cultural events using IoT, analytics, and automation.

## Modules
1. **Smart Lighting** – IoT-based system for optimized electricity use.  
2. **Eco-Composting** – Sensor-driven waste management and compost tracking.  
3. **Attendee Tracking** – GPS-enabled crowd monitoring for safety.  
4. **Dashboard** – Unified interface visualizing data in real time.

## Sprint 1 Achievements
- 25–27% energy reduction achieved.
- 55% compost tracking operational.
- 92% tracking accuracy for attendees.
